# todoable-api
Interview Test - Ruby wrapper around the http://todoable.teachable.tech/ API

We use httparty for API calls and Rspec for testing.

# Setup 

`gem install httparty`

`gem install rspec`

# Usage
`require 'todoable_api'`

# Test
`cd todoable-api`

`rspec tests/todoable_api_spec.rb`
